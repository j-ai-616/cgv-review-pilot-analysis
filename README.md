# CGV 영화 리뷰 데이터 기반 관객 반응 분석

> CGV 영화 리뷰 데이터를 바탕으로 관객 반응의 감성 분포, 시계열 변화, 핵심 표현 구조, 그리고 BERTopic 기반 토픽 구조를 분석한 Streamlit 프로젝트입니다.  
> 영화 소비자 반응을 데이터 기반으로 해석하고, 이를 시각적으로 전달하는 포트폴리오형 분석 프로젝트입니다.

## 📖 프로젝트 개요

온라인 영화 리뷰는 단순한 감상문을 넘어,  
관객의 기대, 만족, 몰입, 실망, 비교 평가가 집약된 디지털 텍스트 데이터입니다.

본 프로젝트는 CGV 영화 리뷰 데이터를 수집·정제한 뒤, 다음 질문에 답하고자 시작되었습니다.

- 관객 반응은 전체적으로 얼마나 긍정적인가?
- 반응은 시간에 따라 어떻게 변화하는가?
- 긍정 리뷰와 부정 리뷰는 각각 어떤 표현을 중심으로 형성되는가?
- 관객 반응은 어떤 주제(Topic) 구조로 나뉘는가?

즉, 본 프로젝트는 단순 평점 확인을 넘어  
**디지털 플랫폼 위에서 형성되는 집단적 관객 반응 구조**를 데이터로 읽어보려는 시도입니다.

## 📊 핵심 분석 결과

### 1. 감성 분포

- 총 리뷰 수: **46,599건**
- 긍정 리뷰 수: **45,265건**
- 부정 리뷰 수: **1,334건**
- 긍정 비율: **97.14%**

➡️ 전체적으로 매우 긍정적인 관객 반응이 형성되었습니다.

---

### 2. 시계열 변화

- 날짜별 리뷰 수 변화 확인
- 날짜별 긍정 비율 추이 확인
- 후반부 일부 구간에서 긍정 비율 소폭 하락 관찰

➡️ 영화 평가는 고정된 값이 아니라,  
입소문과 누적 기대에 따라 시간적으로 변화할 수 있음을 보여줍니다.

---

### 3. 긍정 리뷰 핵심 표현

- 배우 연기
- 감동
- 여운
- 몰입감
- 최고
- 재미있다

➡️ 작품의 **정서적 설득력**과 **배우 퍼포먼스**가 긍정 평가 핵심 축으로 나타났습니다.

---

### 4. 부정 리뷰 핵심 표현

- 스토리
- 연출
- 기대 대비 아쉬움
- 지루함
- CG
- 개연성

➡️ 부정 평가는 작품 전체 부정보다  
특정 요소에 대한 세부 평가 형태로 나타났습니다.

## 🧠 BERTopic 분석 결과

BERTopic 기반 토픽 모델링 결과, 관객 반응은 다음 4개 주요 담론 축으로 구성되었습니다.

| Topic | 내용 |
|------|------|
| Topic 0 | 배우 연기와 감동 중심의 전반적 호평 |
| Topic 1 | 유해진·박지훈 연기에 대한 집중 호평 |
| Topic 2 | 오랜만에 만족스러운 한국영화라는 반응 |
| Topic 3 | 호랑이 CG에 대한 아쉬움 |

➡️ 관객 반응은 단순 긍정·부정 이분법이 아니라,  
여러 평가 축으로 동시에 구성되어 있음을 확인했습니다.

## 🧠 해석 가능 이론

### 기대불일치 이론

관객은 관람 전 이미 기대를 형성합니다.  
실제 경험이 기대에 못 미치면 실망이 발생할 수 있습니다.

➡️ 일부 부정 리뷰는 작품 자체보다  
**높은 기대 대비 아쉬움**일 가능성이 있습니다.

### 사회적 증거 / 밴드왜건 효과

화제성, 추천, 입소문은 후속 관객 기대 수준을 높일 수 있습니다.

➡️ 평가는 영화 자체뿐 아니라  
**사회적으로 형성된 기대 수준**의 영향도 받습니다.

### 플랫폼 리뷰 데이터의 의미

온라인 리뷰는 자발적으로 생성된 대규모 텍스트 데이터입니다.

➡️ 설문조사보다 현실적인 감정과 표현 구조를 보여줍니다.
